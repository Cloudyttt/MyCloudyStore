# MyCloudyStore
An Android project -- Cloudy Store, personal work for Android course in 2017.

系统功能描述：
---------------------------------------
>用户账号功能
<ol>
<li>登录注销功能</li>
<li>用户注册功能</li>
<li>修改用户信息功能</li>
<li>登录时记住密码功能</li>
<li>上述操作过程中的用户数据库表增删查改操作功能</li>
<li>各类用户在账号操作过程中检错机制</li>
</ol>

>应用商店浏览功能
<ol>
<li>每日应用推荐浏览</li>
<li>应用分类浏览</li>
<li>应用搜索浏览</li>
</ol>
